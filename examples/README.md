# Examples

This folder contains real or representative transcripts showing PAIciente in use across different platforms and scenarios.

## Structure

- Each **subfolder** represents a scenario (e.g., `homework-startup/`, `device-transition/`, `bedtime-routine/`).
- Each **file** within a scenario folder is a platform-specific transcript (e.g., `claude.md`, `chatgpt.md`, `mistral.md`).
- All transcripts follow the schema defined in [`format.md`](format.md).

## Cross-platform comparison

Multiple transcripts for the same scenario on different platforms are intentional. This documents behavioral differences between LLMs when running the same system prompt — including variations in persona fidelity, defiance realism, clinician structure, and command handling.
