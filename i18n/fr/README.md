# Traductions — Français (FR)

Ce dossier contient les traductions en français de composants sélectionnés de PAIciente.

## Notes importantes

- L'**anglais (EN)** est la langue canonique du projet. Toute la documentation de référence se trouve dans `/docs/` et `/references/`.
- Les traductions contenues dans ce dossier sont des contributions de la communauté et peuvent ne pas être à jour avec la dernière version de la documentation en anglais.
- Le **system prompt ne nécessite pas de traduction**. PAIciente répond automatiquement dans la langue utilisée par l'utilisateur — la gestion linguistique est native au fonctionnement du modèle.

## Conventions de traduction

- ADHD = TDAH (Trouble du Déficit de l'Attention avec Hyperactivité).
- ODD = TOP (Trouble Oppositionnel avec Provocation).
- Toutes les commandes `@` restent en anglais.
- Les étiquettes des champs `[INNER STATE]` restent en anglais.
- Lors de la première utilisation de termes cliniques, inclure l'équivalent en anglais entre parenthèses lorsque la traduction est ambiguë.

## Comment contribuer

1. Consultez les instructions générales dans [`/CONTRIBUTING.md`](/CONTRIBUTING.md) et les conventions i18n dans [`/i18n/README.md`](/i18n/README.md).
2. Créez le fichier traduit dans ce dossier, en conservant la structure et les noms de fichiers originaux.
3. Incluez l'en-tête de version i18n comme première ligne du fichier.
4. Soumettez via pull request avec l'étiquette `i18n`.
